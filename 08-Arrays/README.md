# Arrays in Java

## Definition
- An array is a collection of elements of the same data type stored in contiguous memory locations.
- Stores multiple values in a single variable, instead of declaring separate variables.
- 0-Based Indexing , Fixed Size, Static

Example:
```java
int numbers[] = {10, 20, 30, 40, 50};
```


## Advantages
- Fast access using index
- Easy traversal of elements
- Efficient memory usage
- Useful for storing multiple values of the same type


## Disadvantages
- Fixed size
- Insertion and deletion are costly
- Stores only same data type elements


## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Access    | O(1) |
| Search    | O(n) |
| Update    | O(1) |
| Traversal | O(n) |
